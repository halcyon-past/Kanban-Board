## How to Run

1. Clone this repository
    ```bash
    git clone <repository-link>
    ```
2. Go inside the folder
    ```bash
    cd Kanban-Board
    ```
3. Install All Dependencies
    ```bash
    npm install
    ```
4. Run the Development Server
    ```
    npm run dev
    ```
